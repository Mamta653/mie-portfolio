# Mamta's Portfolio

> A modern, responsive personal portfolio website built with React, TypeScript, and Tailwind CSS.

## ✨ Live Demo

🔗 **[https://mie-portfolio-iota.vercel.app](https://mie-portfolio-iota.vercel.app)**

---

## 📋 Sections

- **Hero / About** — Introduction and personal summary
- **Projects** — Showcase of featured work
- **Tech Stack** — Technologies and tools I work with
- **Resume** — Downloadable resume
- **Contact** — Get in touch
